Sobre el proyecto

En este proyecto se ha implementado una aplicación sencilla, que consta de una pantalla de login, la cual esta maquetada a partir de la captura de pantalla disponible para dispositivos móviles. Esta pantalla cuenta con validacion y control de formularios mediante Reactive Forms del Framework Angular. El proyecto contiene los siguientes ejemplos de desarrollo:

Maquetación:

  • El proyecto ha sido maquetado a partir de una captura de pantalla usando para ello SCSS como preprocesador de CSS y HTML como lenguaje de marcado.
  • La maquetación cumple con el diseño responsive y tiene un breakpoint de 960px.

    Validación de formularios:

  • Todos los campos de formulario de login seran validados con las siguientes reglas.
  • El campo de email no puede estar vacio.
  • El campo de email debe de coincidir con la expresión regular ^([a-zA-Z0-9_\-\.]+)@([a-zA-Z0-9_\-\.]+)\.([a-zA-Z]{2,5})$.
  • El campo password no puede estar vacio.
  • El campo password debe de tener una longitud minima de 5 caracteres.
  • El formulario cuenta con un switch para recordar los datos de usuario. El switch es de tipo booleano y opcional.
  • El botón de aceptar siempre esta activo, si el formulario contiene errores estos aparecen al pulsar el botón, en caso contrario se ejecuta la acción de login y muestra por pantalla los datos introducidos.

    Test:

  • Test unitarios.
  • Test E2E.
  • Test de componentes.

Pila tecnológica

Ionic es un framework que actua como wrapper para unir tecnologías que permiten la creación de aplicaciones multiplataforma, mediante el uso de tecnologías web. Por ello Ionic permite desarrollar aplicaciones para escritorio mediante Electron, y para móviles mediante Cordova o Capacitor y para web mediante Angular, React o Vue. En este caso el proyecto esta orientado a tecnologías móviles, por lo que la pila tecnologica de este esta compuesta por los siguientes frameworks:

  • Ionic
  • Angular
  • Cordova
  • Node

Test

El proyecto lleva implementado una serie de test en el framework Jassmine. Para lanzar los test y comprobar que todo el proyecto funciona de forma correcta solo es necesario lanzar el siguiente comando:

ng test

Ejecutar el proyecto

Para ejecutar este proyecto es necesario que el equipo cuente con las siguientes tecnologías instaladas:

  • Node.js
  • Ionic-CLI

Una vez instaladas estas tecnologías, solo es necesario clonar este repositorio al equipo donde va a ejecutarse. Con un terminal ha de acceder al directorio raíz del proyecto y ejecutar el siguiente comando para instalar los paquetes necesarios:

npm install

Una vez los paquetes estan instalados es puede optar por ejecutar el proyecto mediante un navegador o mediante un dispositivo movil fisico o emulado. Para la ejecución mediante navegador solo es necesario que se ejecute el siguiente comando:

ionic serve

Para la ejecución en dispositivo fisico o emulado es necesario una configuración extra, así como la instalación de una aplicación de desarrollo para Android o iOS y la adecuada configuración del equipo, para mas información puede seguir este enlace para dispositivos Android y este para dispositivos iOS.
